# Role Clarity Matrix

Operational variance often stems from unclear ownership.

A RACI structure was implemented across all markets.

---

## Example Role Matrix

| Function              | Regional Ops | GM | Finance | Compliance | Corporate |
|-----------------------|-------------|----|---------|------------|-----------|
| Revenue Reporting     | A           | R  | C       | -          | C         |
| Labor Scheduling      | R           | A  | -       | -          | C         |
| Licensing Compliance  | C           | R  | -       | A          | C         |
| Vendor Contracts      | R           | C  | A       | -          | C         |
| KPI Dashboard Review  | A           | R  | C       | -          | C         |

Legend:
R = Responsible  
A = Accountable  
C = Consulted  

---

## Strategic Impact

Clear ownership reduced escalation latency and decision ambiguity across expanding markets.
